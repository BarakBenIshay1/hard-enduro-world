# Slug Policy

Public slugs are permanent by default.

Examples:

- `/events/sea-to-sky-2026`
- `/riders/manuel-lettenbichler`
- `/teams/red-bull-ktm`
- `/manufacturers/ktm`
- `/motorcycles/ktm-300-exc-2026`
- `/countries/spain`

Display names may change over time, but slugs should not change automatically. If an editor intentionally changes a slug, the old value must be saved in `SlugRedirect` so existing links continue to work.
