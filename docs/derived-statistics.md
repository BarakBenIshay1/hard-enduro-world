# Derived Statistics

Derived tables are optimized read models generated from raw results.

Examples:

- `RiderCareerSeason`
- `MotorcycleSeasonStat`
- `ManufacturerSeasonStat`
- `RiderComparisonSnapshot`
- `ChampionshipRecord`
- `HallOfFameEntry`

These records should be recalculated after imports or approved manual edits. They should never replace raw official timing and final classification data as the source of truth.
